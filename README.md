## Run Backend

Go to the terminal and hit the given codes 

  cd backend
    
  pip install -r requirements.txt
    
  python -m uvicorn app.main:app --reload




## Run Frontend 

Open second terminal(Not the one in which backend is running)

  cd frontend

  npx serve .


After that signup with your credentials,

Once sign up is done, login with the same creds

Start learning
